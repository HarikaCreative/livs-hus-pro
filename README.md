# Liv's Hus - Pro Tier

A luxury villa website for Liv's Hus in Apokoronas, Crete - Pro package with premium animations and advanced features.

## ✅ Security Update

- **Next.js 15.1.9** - Patched against CVE-2025-55182 (Critical RCE vulnerability)
- Secure and production-ready

## Design Philosophy

Inspired by luxury villa platforms:
- Villas of Distinction
- Marriott Homes & Villas
- onefinestay
- Landfolk

### Premium Features

- **Framer Motion Animations**: Smooth page transitions and scroll effects
- **Advanced Lightbox**: Category labels and enhanced navigation
- **Booking Calendar Modal**: Interactive availability checker (placeholder)
- **Parallax Hero**: Sophisticated scroll-based effects
- **Staggered Animations**: Professional entrance animations
- **Hover Effects**: Micro-interactions throughout

## Tech Stack

- Next.js 15.1.9 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- **Framer Motion** - Advanced animations
- Lucide Icons

## Installation

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
npm start
```
